# CMR Print Manager - Development Continuation Prompt

You are working on **CMR Print Manager**, a .NET 9 Windows Forms application for printing CMR (Lettre de Voiture) transport documents with support for all 24 standardized CMR places.

## Current Project Status

### ✅ Completed Features
- **24 CMR Places Support** - All standardized fields mapped with default positions
- **Visual Preview** - Left panel shows miniature CMR layout with numbered positions
- **Resizable Text** - Font size control (6-20pt) per field
- **Position Adjustment** - Vertical offset (+/-100px) for alignment
- **Template System** - Save/load/delete layout configurations to `%AppData%\CMRPrint\Templates\`
- **Customer Management** - Add, edit, view customers
- **Excel Import** - Bulk import customers from Excel files (ClosedXML)
- **Print Preview & Printing** - Full preview and printing to any Windows printer
- **Self-contained Release EXE** - 53MB standalone executable (no runtime dependencies)

- `Program.cs` - Entry point

## Key Technical Details

### 24 CMR Places Definition
Located in `CmrPlaces.cs`:
```csharp
public class CmrField
{
    public int PlaceNumber { get; set; }
    public string FieldName { get; set; }
    public string Description { get; set; }
    public float DefaultX { get; set; }
    public float DefaultY { get; set; }
    public int DefaultFontSize { get; set; }
}
```

Each place has a field name (e.g., "ConsignorName"), description, and default print coordinates.

### CmrFieldControl
Custom UserControl (385×50px) containing:
- **Label:** Place description + number
- **TextBox:** User input
- **NumericUpDown (Font Size):** 6-20pt
- **NumericUpDown (Offset):** -100 to +100px

### Template System
- **File Format:** XML (via XmlSerializer)
- **Storage:** `%AppData%\CMRPrint\Templates\`
- **Persistence:** Saves FontSizes and VerticalOffsets dictionaries
- **Scope:** Per-template customization

### Print Logic
**PrintPage handler:**
1. Gets all 24 field values from CmrFieldControl instances
2. Retrieves stored FontSize and Offset for each field
3. Calculates X = DefaultX, Y = DefaultY + Offset
4. Draws text using Font(size=FontSize)
5. Repeats for all 24 places

## Quick Reference: Major Code Flows

### User Flow: Print a CMR
1. User selects customer → `customerComboBox_SelectedIndexChanged()`
2. User fills fields in 24 CmrFieldControl instances
3. User clicks "Preview Print" → `btnPreview_Click()` → `printPreviewDialog.ShowDialog()`
4. PrintPage handler called → `printDocument1_PrintPage()`
5. Iterates through all CmrFieldControl values, gets FontSize/Offset, draws text

### User Flow: Save Template
1. User adjusts font sizes and offsets
2. User clicks "Save" → `btnSaveTemplate_Click()`
3. Prompts for template name
4. Collects all FontSize and VerticalOffset values from 24 controls
5. Calls `TemplateManager.SaveTemplate()` → XML file saved to `%AppData%\CMRPrint\Templates\`

### User Flow: Import Customers
1. User clicks "Import Excel" → `btnImportExcel_Click()`
2. OpenFileDialog opened
3. `ExcelImporter.ImportCustomers()` reads Excel rows
4. For each row with non-empty Name, creates Customer object
5. Adds to `_customers` list
6. Reloads customer combo box
