# CMR 24 Places Quick Reference Guide

## CMR Form Layout Map

This guide shows where each of the 24 numbered places appears on a standard CMR (Lettre de Voiture) document.

### Left Column (Consignor Side)

**PLACES 1-5: SENDER INFORMATION**
```
┌─────────────────────────────────────┐
│  [1] CONSIGNOR NAME & ADDRESS       │
│      Full legal name, street address │
│      postal code, city               │
├─────────────────────────────────────┤
│  [2] CONSIGNOR PHONE/REFERENCE      │
│      Contact telephone, fax, ref    │
│      order number if applicable      │
├─────────────────────────────────────┤
│  [3] LOADING INSTRUCTIONS           │
│      Special handling notes         │
│      Loading requirements           │
├─────────────────────────────────────┤
│  [4] CONSIGNOR REMARKS              │
│      Additional sender notes        │
│      Hazmat info if needed          │
├─────────────────────────────────────┤
│  [5] DOCUMENTS ATTACHED             │
│      ☑ CMR  ☐ Invoice  ☐ Other     │
└─────────────────────────────────────┘
```

**PLACES 6-7: TRANSPORT DETAILS (HEADER ROW)**

┌──────────────┬──────────────┬──────────────────┐

## How to Use Each Place in the Application

### Step 1: Basic Shipment Info (Places 1-2)
- Place [1]: Enter your company name, address, postal code, city
- Place [2]: Your telephone number, email, order reference number

### Step 2: Handling Instructions (Places 3-5)
- Place [3]: Any special loading needs ("Handle with care", "Fragile", etc.)
- Place [4]: Additional notes for the shipper
- Place [5]: Check which documents are attached (CMR always included)

### Step 3: Goods Description (Places 6-10)
- Place [6]: Mark if seals have been applied
- Place [7]: Packaging type: "Cartons", "Pallets", "Crates", etc.
- Place [8]: Detailed description of goods (most important field!)
- Place [9]: Total weight in kg
- Place [10]: Your internal order number or manifest reference

### Step 4: Transport Details (Place 11)
- Place [11]: Where freight charges are payable (e.g., "Consignor", "Consignee", "Third Party")

### Step 5: Recipient Info (Places 12-14)
- Place [12]: Recipient company name, address, postal code, city
- Place [13]: Recipient phone number, contact person, order number
- Place [14]: Special unloading instructions for the delivery point

### Step 6: Observations (Places 15-16)
- Place [15]: Filled by driver during transport (damage notes, observations)
- Place [16]: Filled by consignee upon receipt (acceptance, condition)

### Step 7: Administrative (Places 17-19)
- Place [17]: Vehicle registration, driver license, transport authorization
- Place [18]: Route through countries (e.g., "Germany > France > Belgium")
- Place [19]: If insured: insurance company name and policy number

### Step 8: Signatures (Places 20-24)
- Place [20]: Your authorized signature and company stamp
- Place [21]: Date of transport (when pickup occurs)
- Places [22-24]: Reserved for shipper, carrier, and receiver signatures

## Field Sizing Tips

- **Font too small?** Increase the font size value in the app (6-20pt)
- **Text not aligned?** Use the Offset value to move text up (-) or down (+)
- **Using non-standard CMR forms?** Create a template to save your adjustments

## Common Issues & Solutions

| Problem | Solution |
|---------|----------|
| Text runs off page | Reduce font size or shorten text content |
| Text doesn't print at right place | Adjust offset value and preview again |
| Can't fit all goods description | Use abbreviated text or Place [14] notes |
| Special characters not printing | Stick to basic ASCII (no special symbols) |

**Quick Stats:**
- **Total Fields:** 24
- **Required Fields:** At minimum [1], [8], [9], [12], [20], [21]
- **Character Limit per field:** No fixed limit (adjust font size as needed)
- **Paper Size:** Standard A4 (210 × 297 mm)
